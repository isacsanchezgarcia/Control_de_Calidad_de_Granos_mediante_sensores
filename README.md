# Control_de_Calidad_de_Granos_mediante_sensores
Sistema de control de calidad de los granos de soja que mide y analiza la temperatura y la humedad mediante sensores y un programa en C++ basado en POO. Con ello, busca evitar pérdidas de calidad y deterioro del grano, optimizando el proceso de conservación y asegurando un control eficiente y automatizado.
